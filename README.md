# Java Problem-Solving Exercises

A collection of Java programs implementing common programming challenges,
focused on logic building, input handling, and algorithmic thinking.

## Programs

`EvenArrayOddArray` | Separates an integer array into even and odd elements 
`Marks` |  Calculates student averages and tracks pass/fail thresholds
`PasswordValidation` | Validates passwords against defined security rules 
`PyramidOfNumbers` | Generates numeric pyramid patterns using nested loops 
`ReverseInput` | Reverses user-provided number 

## Concepts Practiced
- Array manipulation
- Data processing
- Counting logic
- Pattern generation with loops
- Modulus operator
- Separating logic into reusable methods

## Language
Java
